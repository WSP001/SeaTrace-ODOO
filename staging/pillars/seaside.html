SeaSide (HOLD)

Vessel tracking and initial data capture

Overview

SeaSide is the foundation of the Four Pillars architecture. It captures initial vessel data, tracks locations in real-time, and provides the "HOLD" checkpoint for all seafood traceability records.

Key Features:

  • Real-time AIS vessel tracking
  • GPS-verified positioning
  • Vessel registration and certification
  • Initial quality scoring
  • Fishing zone compliance

API Endpoints

GET /api/v1/seaside/vessels

Retrieve list of registered vessels with real-time tracking data

{
  "success": true,
  "count": 6,
  "data": [
    {
      "id": "vessel-001",
      "name": "Pacific Star",
      "imo": "9123456",
      "mmsi": "367123456",
      "location": {
        "lat": 55.3,
        "lon": -166.2,
        "timestamp": "2025-10-20T14:30:00Z"
      },
      "status": "Fishing"
    }
  ]
}

GET /api/v1/seaside/vessels/{id}

Get detailed information for a specific vessel

{
  "id": "vessel-001",
  "name": "Pacific Star",
  "operator": "Bering Sea Fisheries Co-op",
  "certifications": ["MSC-F-12345", "RFM", "BAP"],
  "targetSpecies": ["Alaska Pollock", "Pacific Cod"]
}

POST /api/v1/seaside/vessels/{id}/positions

Update vessel position (requires authentication)

{
  "lat": 55.3,
  "lon": -166.2,
  "speed": 4.2,
  "heading": 245,
  "timestamp": "2025-10-20T14:30:00Z"
}

POST /api/v1/seaside/quality/score

Submit initial quality assessment (requires authentication)

{
  "vesselId": "vessel-001",
  "temperature": 1.5,
  "handlingMethod": "Flash frozen at sea",
  "score": 99
}

Use Cases

For Vessel Operators:

  • Register vessels and maintain certifications
  • Submit real-time position updates
  • Record initial catch quality data

For Compliance Teams:

  • Monitor vessel locations vs. fishing zones
  • Verify certification validity
  • Audit GPS-verified records
