MarketSide (EXCHANGE)

Consumer verification and market integration

Overview

MarketSide provides the "EXCHANGE" checkpoint where seafood products are verified at point-of-sale, enabling consumers to validate authenticity and sustainable sourcing claims.

Key Features:

  • Consumer-facing QR code verification
  • Full supply chain visibility
  • Market transaction recording
  • Pricing and landed cost tracking
  • Sustainability certification display
  • Commons Fund transparency
üí∞ MarketSide Funds the Commons: 10% of enterprise API revenue supports public infrastructure, ensuring free access to SeaSide, DeckSide, and DockSide for local fisheries worldwide.

Consumer Use Case

How it works:

  1. Consumer scans QR code on seafood package
  2. Browser opens verification page
  3. Full supply chain displayed:
    • Vessel name and location
    • Catch date and GPS coordinates
    • Processing facility and temperature logs
    • Certifications (MSC, BAP, etc.)
    • Sustainability score
  4. Consumer can verify all claims instantly
